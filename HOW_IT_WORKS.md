# How It Works

SkySight состоит из наземной станции, бортового модуля, симулятора Unreal Engine и вспомогательных сервисов. Основной сценарий: камера БПЛА передаёт кадры, модель находит опасные объекты, система переводит их в координаты карты, подтверждает событие и показывает оператору.

## роли

| роль | назначение |
|---|---|
| `ground` | графическая наземная станция оператора |
| `module` | бортовой headless-модуль с детекцией, телеметрией и маршрутизацией |

## поток данных

1. БПЛА или Unreal Engine отдаёт видеопоток и телеметрию.
2. Бортовой модуль запускает YOLO-инференс.
3. Система сглаживает bbox и фильтрует слабые предсказания.
4. GeoProjector переводит координаты bbox в координаты карты.
5. Агрегатор подтверждает цель через K/N voting.
6. Реестр объектов объединяет близкие цели и подавляет повторы.
7. Наземная станция показывает событие оператору.
8. Маршрутизатор может построить облёт цели или вернуть БПЛА на маршрут.

## бортовой модуль

Бортовой модуль отвечает за обработку данных рядом с источником видео. В реальном сценарии он может работать на Jetson, мини-ПК или другом вычислителе, связанном с дроном.

Основные задачи:

- получение кадров с камеры;
- запуск YOLO-модели;
- фильтрация классов;
- сглаживание bbox между кадрами;
- пересчёт bbox в географическое положение;
- подтверждение целей через агрегацию;
- передача результата в наземную станцию;
- корректировка маршрута при появлении цели.

## наземная станция

Наземная станция показывает карту, видеопоток, телеметрию, найденные объекты и журнал событий. Оператор может оценить тревогу, подтвердить объект, отклонить ложное срабатывание или отправить дрон на облёт.

## детекция и агрегация

| уровень | назначение |
|---|---|
| confidence threshold | отсекает слабые предсказания модели |
| IoU threshold | управляет NMS и пересечениями bbox |
| bbox smoothing | уменьшает дрожание рамок |
| K/N voting | подтверждает цель только после нужного числа попаданий |
| geo deduplication | объединяет близкие цели в одну запись |
| suppression TTL | временно подавляет повторные тревоги в той же зоне |

## маршрутизация

Маршрутизатор строит маршрут обследования и может добавлять манёвры вокруг подтверждённых целей. В проекте заложены идеи grid / lawn-mower маршрута, TSP-оптимизации, учёта батареи, no-fly зон и orbit-облёта.

## интеграции

| backend | назначение |
|---|---|
| `unreal` | HTTP-мост к Unreal Engine симулятору |
| `stub` | локальная имитация телеметрии для разработки |
| `mavlink` | подключение реального БПЛА через MAVLink |
| `custom` | внешний SDK через integration service |
