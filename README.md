## о проекте

**SkySight** — программный комплекс для мониторинга лесных и труднодоступных территорий с помощью БПЛА. Система получает видеопоток с дрона или симулятора, выполняет нейросетевую детекцию пожара, дыма и людей, переводит найденные объекты в координаты карты и показывает результат оператору в наземной станции.

Проект объединяет наземную станцию, бортовой модуль, маршрутное планирование, REST API и симулятор на Unreal Engine 5. Благодаря этому систему можно демонстрировать и тестировать без реального дрона, а затем подключать к реальному БПЛА через MAVLink или отдельный SDK.

## ключевые возможности

- обнаружение пожара, дыма и людей на кадрах с камеры БПЛА;
- отображение найденных объектов на карте;
- наземная станция оператора с видеопотоком, телеметрией, маршрутом и журналом событий;
- фильтрация ложных тревог через агрегацию и дедупликацию;
- построение маршрутов обследования территории;
- автоматический облёт подтверждённых целей;
- симулятор на Unreal Engine 5 для тестирования сценариев;
- поддержка локального стаба, Unreal backend, MAVLink и внешнего SDK;
- развёртывание на рабочей станции, сервере, ARM-устройстве или Jetson.

## состав системы

| компонент | назначение |
|---|---|
| **Ground Station** | интерфейс оператора: карта, видео, события, телеметрия и управление миссией |
| **Onboard Module** | обработка кадров, запуск модели, геопроекция и передача подтверждённых событий |
| **Detection Pipeline** | YOLO-инференс, сглаживание bbox, агрегация и реестр объектов |
| **Route Planner** | построение маршрутов обследования и облёта целей |
| **Unreal Simulator** | виртуальная среда с дроном, камерой, огнём, дымом и управляемой телеметрией |
| **REST API** | интеграция модулей, приём детекций, статус системы и метрики |

## сценарий работы

1. Оператор выбирает район обследования.
2. Система строит маршрут для БПЛА.
3. Дрон или симулятор отдаёт видеопоток и телеметрию.
4. Модель обнаруживает пожар, дым или людей.
5. Система подтверждает событие и переводит его в координаты карты.
6. Оператор видит объект, оценивает ситуацию и при необходимости запускает облёт цели.

## назначение

SkySight может использоваться как демонстрационный, учебный или исследовательский комплекс для разработки систем мониторинга с БПЛА. Проект подходит для проверки алгоритмов обнаружения, тестирования сценариев в симуляторе и подготовки к подключению реального дрона.
